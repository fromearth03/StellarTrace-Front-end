# Stellar Trace Frontend

A sci‑fi themed search UI for exploring academic articles, with animated views and an add‑document flow.

## Features
- Observatory search view
- Results orbital view
- Article detail overlay
- Add document form
- 
## Tech Stack
- React
- Framer Motion
- Tailwind CSS

## Getting Started
```bash
npm install
npm run dev
```

## Lint
```bash
npm run lint
```
